0 -> binary digit -> bit
1 -> binary digit -> bit
01000001 (A) -> bity

um bity é um conjunto de oito bit´s

o computador só entendo bit´s e bity´s

1024 bytes = 1 KB -> 1 KiloBity

1024 KB = 1 MB -> 1 MegaBity

1024 MB = 1 GB -> 1 GibaBity

1024 GB = 1 TB -> 1 TeraBity

1024 TB = 1 PB -> 1 PetaBity     

1024 PB = 1 EB -> 1 ExaBity

1024 EB = 1 ZB -> 1 ZetaBity

10214 ZB = 1 YB -> 1 YotaBity

Colocar a letra 'B' minuscula faz diferença.

MB -> MegaBity
Mb -> Megabits 

Usasse MegaBity/GigaBity/TeraBity/etc.. para representar armazenamento.
ex: Comprei um pendrive de um giga (GigaBity)

Usasse MegaBits/GigaBits/Terabits/etc... para representar transmissão
ex: Minha internet é de 200 megas (200 MegaBits)

HTML é uma linguagem de marcação

CSS ->  Cascading Style Sheets

HTML é uma linguagem focada em conteúdo -> Textos

                                                                                      Vídeos

CSS é uma linguagem focada em design -> Cores
                                                                           Sombras
                                                                           Tamanhos 
                                                                           Posicionamento

                                                                                              Animações
                                                                                              Popups
                                                                                               Validações
